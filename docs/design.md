# Picking Mini
## Vision

「薬局の現場知識を蓄積し、誰でも同じ品質で薬を探せるナレッジベースを作る。」

## 1. コンセプト

薬局で薬を探す時間をゼロにする。

- 迷ったら検索
- 無かったらその場で登録
- 使いながら育つ薬局内データベース

---

## 2. システム構成

```text
GitHub Pages
  ↓
HTML / JavaScript
  ↓
Google Apps Script API
  ↓
Google Spreadsheet
````

---

## 3. MVP

* 薬品検索
* 場所表示
* 新規登録
* 編集
* 写真URL表示
* 最近検索した薬

---

## 4. 画面仕様

### 4.1 検索画面

* 薬品名・一般名・略称で検索
* 入力中に候補表示
* 検索結果はカード形式
* 見つからない場合は新規登録へ誘導
* 最近検索した薬を最大20件表示

### 4.2 詳細画面

表示項目：

* 薬品名
* 場所
* 棚写真
* 一般名
* 別名
* メモ

操作：

* 編集
* 写真更新
* お気に入り

### 4.3 新規登録画面

最小入力：

* 薬品名
* 場所
* メモ

後から追加入力：

* 一般名
* 別名
* 写真URL
* 分類

---

## 5. 検索仕様

検索対象：

* displayName
* genericName
* aliases
* location
* note

例：

* ロキソ → ロキソニン
* エプレ → セララ
* アムロ → アムロジン / ノルバスク
* デュピ → デュピクセント
* アスピリン → バイアスピリン

---

## 6. データ構造

シート名：DrugMaster

| column      | description |
| ----------- | ----------- |
| id          | 一意ID        |
| displayName | 表示名         |
| genericName | 一般名         |
| aliases     | 別名・略称・検索語   |
| location    | 置き場所        |
| note        | メモ          |
| imageUrl    | 写真URL       |
| favorite    | お気に入り       |
| createdAt   | 作成日時        |
| updatedAt   | 更新日時        |

---

## 7. 写真機能

### 目的

薬品の置き場所を視覚的に確認できるようにする。

### MVP

* 登録時に写真URLを保存できる
* 詳細画面で写真を表示する

### 将来

* スマホから写真アップロード
* Google Drive保存
* 棚全体写真と薬品位置メモ

## 8. API

### POST save
